Phenox はユーザーがプログラムを書き換えることで、独自の挙動を実行することができます。しかし、周辺環境の状態によっては予期しない動作を行うことがあります。この章では Phenox を安全に使うために必要な事項を説明してきます。実機に触れながら理解を深めて、読み進めていってください。
